Wave Agent X - User Guide

Getting Started

Import Files

Click the Import button or drag folders/files into the File List to add audio files to your session. Wave Agent X supports WAV, MP3, and AAC formats with 16, 24, or 32-bit float bit depths.

Note that files larger than 2GB may have limited support depending on your browser and system.

You can import individual files or entire directories. The app will recursively scan folders and add all compatible audio files. All metadata is preserved during import.

Managing Your Project

Once files are imported, you'll see them in the metadata table below. Select individual files to load them into the waveform and mixer for playback and editing. The app tracks which file is currently loaded and maintains session state across your work.

First Time Setup Tips

  • Enable auto-save in the header to automatically save metadata changes
  • Customize your column layout by dragging headers to show the fields you care about most
  • Check the User Guide frequently—Wave Agent X has many features!

Metadata Management

Understanding Metadata Fields

Wave Agent X manages the following metadata fields:

  • Filename: The name of the audio file
  • Project: Project or production name
  • Tape: Tape or media identifier
  • Channels: Number of audio channels (read-only)
  • Bit Depth: Audio bit depth in bits (read-only)
  • Sample Rate: Sample rate in Hz (read-only)
  • Format: Audio format (WAV, MP3, etc.)
  • Scene: Scene name or identifier
  • Take: Take number
  • Duration: File length in time format (read-only)
  • Start TC: Timecode at file start
  • End TC: Timecode at file end (read-only)
  • FPS: Frames per second for timecode
  • File Size: File size in MB/GB (read-only)
  • Notes: Notes about the audio file

Editing Metadata

Click on any editable cell in the metadata table to modify it. Changes are auto-saved by default (if enabled). For fields with special formats like timecode, the app will validate your input.

  • Click once to enter edit mode
  • Press Enter to save and move to the next field
  • Press Escape to cancel editing

Batch Edit Operations

Select multiple files using Ctrl/Cmd+Click or Shift+Click for ranges. Then click Batch Edit to open a menu where you can apply changes to all selected files at once:

  • Check the checkbox for each field you want to modify
  • Enter values that will be applied to all selected files
  • Click OK to apply or Cancel to dismiss

Column Management

You can customize which columns appear and in what order:

  • Drag column headers to reorder them
  • Your layout is saved in browser storage for the next session
  • Focus on the metadata fields most important to your workflow

Exporting & Saving

Use the Save button to manually save metadata to files (when auto-save is disabled). Click Export to create and save a stereo or mono mixdown of the selected file(s).

Waveform and Playback

Playback Controls

Select a file from the table and use the playback controls in the Player section:

  • ▶ (Play): Start playback from the current position
  • ⏹ (Stop): Stop playback and reset to the beginning
  • ↻ (Loop): Toggle loop mode. When looping a region, playback will repeat between the region boundaries

Time Display & Navigation

The time display shows the current playback position in both:

  • Elapsed Time: HH:MM:SS.MS format (hours, minutes, seconds, milliseconds)
  • Timecode: HH:MM:SS:FF format (hours, minutes, seconds, frames) based on the file's FPS setting

Click anywhere on the waveform to jump to that position. The playhead (vertical line) shows your current position in the audio.

Waveform Display

The waveform shows the amplitude of each audio channel:

  • Each channel is color-coded for easy identification
  • Soloing a channel only shows that channel's waveform
  • The amplitude scale on the left shows dBFS levels (-∞ to 0)
  • Muted channels don't affect the waveform display (visual only—use the mixer to control audio)
  • For files larger than 2GB, the waveform display is not available due to browser memory limitations

Creating and Managing Regions

Regions allow you to select a specific portion of audio for export or processing:

  • Create: Click and drag on the waveform area (between the time display and waveform)
  • Resize: Click and drag the left or right edge to adjust boundaries
  • Move: Click and drag anywhere in the middle of the region to move it
  • Clear: Click anywhere on the waveform outside the region

The region is highlighted in blue with transparency. When you export audio, if a region is defined, only that section will be exported.

Working with Cue Markers

Cue markers help you navigate and mark important points in your audio:

  • Add Marker: Press M to add a marker at the current playback position
  • Move Marker: Click and drag a marker on the waveform to relocate it
  • Name Marker: Double-click a marker to open a dialog where you can name it
  • Delete Marker: Select a marker (click once) and press Delete
  • Navigate Markers: Press < to jump to the previous marker (or start if there is no previous marker) or > to jump to the next marker (or end if there is no next marker).

Markers are saved in the file's cue chunk, so they persist between sessions and can be shared with other audio software that supports the cue chunk format. Useful for marking sync points, problem areas, or important phrases.

Region vs. Marker Comparison

Regions: Define a time range (start to end). Used for exporting portions or normalizing specific sections.

Markers: Single points in time. Used for navigation and marking important moments.

Channel Mixer

Understanding the Channel Strips

Each channel in your audio file gets a vertical strip in the mixer with the following controls:

  • Channel (Track) Name: Editable
  • Gain Fader: Vertical slider to control fader gain
  • Pan Knob: Stereo positioning control
  • VU Meter: Real-time level display
  • Gain Display: Shows current gain in dB
  • Mute & Solo Buttons: M and S buttons for channel monitoring. Soloing a channel only shows that channel's waveform too.
  • Automation Controls: A (arm) and × (clear) buttons

Gain Control & Gain Structure

Each channel has a gain control that affects the audio master mix output:

  • 0.0 dB: Unity gain (no change to audio level)
  • Positive dB: Increases gain (e.g., +6 dB)
  • Negative dB: Decreases gain (e.g., -12 dB)
  • -∞ dB: Complete silence (bottom of fader)
  • Drag the fader or double-click to reset to 0.0 dB

The gain field shows current gain in dB. Click to manually enter a gain value.

Panning (Stereo Positioning)

The pan knob controls where each channel appears in the stereo field:

  • C (Center): Channel is equally in both left and right speakers
  • L1-L10: Channel is positioned increasingly to the left
  • R1-R10: Channel is positioned increasingly to the right

Use panning to create stereo imaging or to separate channels. Double-click the pan knob to reset to center.

Mute and Solo Controls

Mute Button (M): Silences the channel completely. Other channels continue to play at their normal levels. Multiple channels can be muted simultaneously.

Solo Button (S): Plays only the soloed channel while muting all others. When you solo a channel, any previously muted channels remain muted. Solo is useful for:

  • Isolating and listening to a single channel
  • Checking for problems or noise on specific channels
  • Monitoring dialogue or specific instruments

Stereo Linking for Paired Channels

When you have stereo pairs (L/R channels), you can link them for coordinated control:

  • Click the link icon between odd and even channels (1-2, 3-4, etc.)
  • Choose linking mode in the dialog that appears:
  • Linked Mode (L/R): Standard stereo linking where L and R channels pan symmetrically. Adjusting the L channel fader affects both the L and R channel fader gains.
  • M/S Mode (Mid-Side): For mid-side encoded stereo. The odd channel fader controls M (mid), the even channel fader controls S (side). Odd channel pan controls LR balance and even channel pan controls stereo width

The link icon shows the current mode. You can unlink channels by clicking the icon again.

Automation Recording & Playback

Capture dynamic changes in volume, panning, or muting:

  • Arm for Recording: Click the A button on a channel to enable automation recording
  • Record Automation: Start playback and move the faders or pan knobs. All movements are recorded with precise timing
  • Playback: Once recorded, automation plays back automatically during subsequent playback
  • Clear Automation: Click the × button to delete all recorded automation for that channel
  • Touch-Write Mode: Automation only records when you're actively touching a control

Workflow example: Create a smooth fade-out by arming the channel, playing back the audio, and gradually moving the volume fader down during the last 5 seconds. The automation will repeat exactly on subsequent playbacks.

VU Meters

The vertical meter next to each fader provides real-time level monitoring:

  • Shows peak levels during playback
  • Helps you identify clipping or distortion
  • Displays the signal chain: Input → Gain → Meter → Pan → Output

Master Channel

The master strip on the far left controls the overall output level of all channels combined. Always keep the master at a reasonable level to prevent clipping.

File Operations

Normalize Audio

Normalize adjusts the volume of audio files to reach a specific target level:

  • Select one or more files
  • Click Normalize to open the normalization dialog
  • Set the target level (default: -1.0 dBFS, standard for broadcast)
  • Choose to normalize the entire file or a selected region

Use case: When you have multiple takes at different recording levels, normalization brings them all to the same loudness for consistency.

Rename Files Batch

Rename multiple files using metadata templates:

  • Select files to rename
  • Click Rename to open the rename dialog
  • Choose or create a naming pattern using metadata fields
  • Preview the new names before applying

Example templates:

  • [Project]_[Scene]_[Take] → MyMovie_S01_T01
  • [Tape]_[StartTC] → Tape01_001030000

Export Audio

Save your processed audio with metadata preserved:

  • If a region is selected, only that section exports
  • Choose output format and settings
  • All metadata is embedded in the exported file
  • Automation is not applied to exports—use the mixer settings instead

Group Siblings

Organize related files (multi-track takes from the same scene):

  • Select files from the same take
  • Click Group Siblings to link them
  • Grouped files can be selected and edited as a unit
  • Useful for batch operations on coordinated takes

Combine Files

Merge multiple audio files into a single file:

  • Select the files you want to combine
  • Click Combine
  • Choose how files are combined (sequentially or as multi-channel)

Split Files

Break a file into multiple parts at markers or regions:

  • Create markers at split points
  • Select the file and click Split
  • New files are created between markers

Conform to CSV

Bulk update metadata from a spreadsheet:

  • Create a CSV file with columns matching your metadata fields
  • Match files by filename or another unique identifier
  • Click Conform to CSV and select your file
  • All matching files get their metadata updated

Extract TC Range

Extract a portion of audio based on timecode range:

  • Specify start and end timecode
  • The tool extracts only that section and saves it as a new file
  • Useful for pulling specific scenes from longer recordings

Sound Report & Diagnostics

Generate reports about your audio files:

  • Report: Creates a summary of all loaded files with their metadata
  • Diagnostics: Shows detailed technical information and checks for potential issues

Tips & Tricks

Workflow Optimization

  • Use keyboard shortcuts extensively: Press M, use < and >, arrow keys. These save time over mouse clicking.
  • Customize your column layout: Show only the metadata fields you actually use in your work. Fewer columns = easier navigation.
  • Batch operations: Whenever possible, do multiple operations at once with batch edit rather than individually.
  • Save frequently: While auto-save is convenient, manually save important changes to ensure they're persisted.

Technical Tips

  • Offline Operation: Wave Agent X works completely offline. Your files never leave your computer. Perfect for on-set work or locations without internet.
  • Service Worker Caching: The first time you load the app, it caches all assets. Subsequent loads are nearly instant. Works even with poor connectivity.
  • Session Persistence: Your mixer settings, column layout, and recently loaded file are saved in browser storage. Close and reopen the app to resume where you left off.
  • Large Files: The app handles files up to 2GB. Very large files >2GB won't show waveforms due to browser memory limits, but you can still edit metadata and export.

Audio Quality

  • Always export as WAV: WAV preserves maximum audio quality and supports metadata standards.
  • Monitor levels carefully: Use the VU meters to ensure you're not clipping. Clipped audio is permanently damaged.
  • Use markers for reference: Mark problem areas, sync points, or important edits so you can quickly navigate back to them.
  • Test automation: Record automation, play it back, and verify it matches your intent before exporting.

Troubleshooting Tips

  • Audio not playing: Check that channels aren't muted (M buttons) and master volume isn't all the way down.
  • Metadata not updating: Ensure auto-save is enabled or manually click Save after editing.
  • Waveform not showing: Files larger than 2GB don't display waveforms. You can still work with metadata and audio.
  • Automation playing strangely: Clear automation and re-record it. Use touch-write mode for more natural results.

Supported Formats

  • WAV: Full support including BWF (Broadcast Wave Format) with bEXT metadata, iXML, and other chunks. RF64 for files >2GB. Recommended format.
  • MP3: Import and playback supported. Metadata limited compared to WAV.
  • AAC: Import and playback supported via browser audio decoder.

Best Practice: Always export as WAV to preserve quality, metadata, and timecode information.

Metadata Standards

Broadcast Wave Format (BWF)

Wave Agent X respects the broadcast audio standard:

  • bEXT chunk: Contains description, originator, originator reference, origination time, and other metadata
  • iXML chunk: Contains XML-formatted metadata like scene, take, tape name
  • cue chunk: Stores marker positions and names

This ensures your metadata is preserved when working with professional audio software.

Troubleshooting

Waveform Not Displaying

Waveforms only display for files under 2GB due to browser memory limitations. For very large files, you can still:

  • Edit metadata
  • Listen to audio (all files are playable)
  • Add markers and regions
  • Export processed audio

Metadata Not Saving

If metadata changes don't persist:

  • Verify auto-save is enabled in the header
  • Manually click Save after editing
  • Check browser storage permissions (may be restricted in private/incognito mode)
  • Ensure your browser isn't clearing storage on exit

Automation Not Playing Back

If recorded automation doesn't play:

  • Verify it was recorded (check the A button is armed while recording)
  • Press play and listen carefully—automation is played automatically
  • Clear automation and re-record if something seems wrong
  • Check that the channel isn't soloed or muted

Audio Playback Issues

  • Check that no channels are muted (look for M button highlighting)
  • Verify master volume isn't all the way down
  • Ensure your computer's volume is turned up
  • Try refreshing the page if audio suddenly stops
  • Check browser console for error messages (F12 → Console)

Import Fails

If files won't import:

  • Verify files are in supported format (WAV, MP3, AAC)
  • Check file size (supported up to 2GB)
  • Ensure file permissions allow reading
  • Try importing one file at a time to isolate problems

Export Issues

If exported files sound wrong:

  • Verify mixer levels before exporting (check VU meters)
  • Check that important channels aren't muted
  • Automation is not applied to exports—use mixer settings
  • For multi-channel files, check channel assignment and panning

About Wave Agent X

Overview

Wave Agent X is a professional audio metadata and file management PWA (Progressive Web App) designed specifically for production sound workflows. It combines waveform editing, multi-track mixing, and comprehensive metadata management in a single, browser-based application.

Key Features

  • Complete Offline Operation: No internet required. Your files stay on your machine.
  • Multi-Track Mixing: Mix, pan, automate, and master multiple channels with professional-quality audio routing
  • Comprehensive Metadata Editing: Manage scene, take, project, timecode, and custom fields
  • Batch Operations: Apply changes to multiple files simultaneously
  • Waveform Visualization: See your audio in real time with accurate amplitude display
  • Cue Markers: Mark important moments and navigate instantly
  • Automation Recording: Capture dynamic fader movements and panning
  • Broadcast Standards: Supports BWF, timecode, and professional metadata formats
  • Service Worker Caching: Fast loading and works with poor connectivity
  • No Installation: Works in any modern web browser

System Requirements

  • Modern web browser (Chrome, Firefox, Safari, Edge)
  • Sufficient storage for your audio files (typically on your computer)
  • About 4GB RAM recommended for comfortable operation with large files

Privacy & Security

Your audio files and metadata are stored locally on your computer. Wave Agent X:

  • Does not upload files to any server
  • Does not track your activity
  • Does not require an account or internet connection
  • All processing happens in your browser using your computer's resources
